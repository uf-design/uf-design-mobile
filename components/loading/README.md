# Loading

#### Props

| 属性               | 说明         | 类型      | 默认值        |
| ------------------ | ------------ | --------- | ------------- |
| **`visible`**      | 显示/关闭    | `boolean` | false         |
| **`label`**        | 提示文字     | `string`  | '加载中...'   |
| **`color`**        | 加载动画颜色 | `string`  | 'white'       |
| **`size`**         | 加载动画大小 | `string`  | 'small'       |
| **`overlayColor`** | 遮罩层背景色 | `string`  | 'transparent' |
