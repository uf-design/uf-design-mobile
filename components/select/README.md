# Select

#### Props

| 属性            | 说明             | 类型       | 默认值 |
| --------------- | ---------------- | ---------- | ------ |
| **`visible`**   | 控制选择器的显示 | `boolean`  | false  |
| **`title`**     | 弹出框标题       | `string`   | '选择' |
| **`options`**   | 可选列表         | `any[]`    | []     |
| **`selected`**  | 当前选中项       | `object`   |        |
| **`renderRow`** | 渲染行结构       | `function` |        |
| **`onChange`**  | 选中回调         | `function` |        |
