# Tips

#### Props

| 属性               | 说明                 | 类型       | 默认值 |
| ------------------ | -------------------- | ---------- | ------ |
| **`visible`**      | 控制 tips            | `boolean`  | false  |
| **`content`**      | 提示内容             | `string`   |        |
| **`contentStyle`** | 内容容器样式         | `array`    |        |
| **`duration`**     | 自动关闭延迟时间     | `number`   | 3000   |
| **`mask`**         | 是否显示遮罩层       | `boolean`  | false  |
| **`onClose`**      | 关闭时出发的回调函数 | `function` |        |
