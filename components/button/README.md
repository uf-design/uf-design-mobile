# Button

#### Props

| 属性           | 说明             | 类型       | 默认值                 |
| -------------- | ---------------- | ---------- | ---------------------- |
| **`size`**     | 按钮大小         | `object`   | {width:150, height:50} |
| **`type`**     | 按钮类型         | `string`   | 'primary'              |
| **`loading`**  | 是否显示加载动画 | `boolean`  | false                  |
| **`disabled`** | 按钮禁用状态     | `boolean`  | false                  |
| **`onPress`**  | touch 事件       | `function` |                        |
