# ListView

#### Props

| 属性            | 说明     | 类型       | 默认值 |
| --------------- | -------- | ---------- | ------ |
| **`fetchx`**    | 数据方法 | `function` |        |
| **`method`**    | 请求类型 | `string`   | 'get'  |
| **`url`**       | 请求地址 | `string`   |        |
| **`params`**    | 请求参数 | `object`   |        |
| **`pageLimit`** | 分页信息 | `number`   |        |
| **`item`**      | 行渲染   | `function` |        |
| **`style`**     | 加载样式 | `object`   |        |
