# Calendar

#### Props

| 属性               | 说明             | 类型       | 默认值 |
| ------------------ | ---------------- | ---------- | ------ |
| **`show`**         | 控制日期选择控件 | `boolean`  | false  |
| **`selectedDate`** | 当前选中日期     | `any`      |        |
| **`onSelect`**     | 确认选择回调     | `function` |        |
